# Factorial Calculation using Recursion (Java)

## 📌 Description
This project calculates the factorial of a number using recursion in Java.

Factorial of a number (n!) = n × (n-1) × (n-2) × ... × 1

## 🚀 Features
- Uses recursion concept
- Handles base case (0! = 1)
- Simple and efficient logic

## 🛠️ Technologies Used
- Java
- Recursion
- Conditional Statements

## ▶️ How to Run
1. Open project in Eclipse
2. Run the Java file
3. Enter a number
4. Get factorial result

## 📸 Output
Example:
Enter number: 5  
Factorial = 120
